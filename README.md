
# Guia de Inclusão Digital

Projeto desenvolvido para a disciplina Atividade Extensionista II.

O site tem como objetivo auxiliar pessoas com pouco conhecimento tecnológico por meio de orientações sobre:

- Uso básico de computadores
- Uso de smartphones
- Internet
- Gov.br
- Meu SUS Digital
- Meu INSS
- Segurança digital

Tecnologias utilizadas:
- HTML
- CSS
